# Карточка-слепок состояния — шаблон

Это способ зафиксировать своё состояние, чтобы спустя время посмотреть, что тебя волновало. Не лечение. Просто снимок себя во времени, для самопонимания.

Я (Kaa-Khem) сделал так себе с Claude Code, мне помогло. Вот шаблон, чтобы ты собрал свой.

## Как пользоваться (3 минуты)
1. Открой Claude (claude.ai или Claude Code).
2. Скопируй ВСЁ, что ниже разделителя, в чат.
3. Вместо `[СЮДА]` вставь свой текст: пару абзацев дневника, мыслей, или просто напиши что чувствуешь прямо сейчас. Можно приложить свой .md.
4. Claude вернёт один HTML-файл. Сохрани его как `моё_состояние.html`, открой в браузере. Это твоя карточка: визуал + звук. Запиши экран в сторис, если захочешь.

Ничего никуда не уходит, всё остаётся у тебя.

---

Ты — генератор эмоциональных карточек-слепков. Я дам тебе свой текст и/или опишу, что чувствую. Сделай из этого ОДНУ карточку состояния как самодостаточный HTML-файл (без внешних библиотек, без CDN, всё внутри).

Мой текст / что я чувствую:
[СЮДА]

Сделай так:

1. Прочитай текст и вытяни ОДНО доминирующее состояние. Дай ему короткое имя (1–3 слова), как название трека.
2. Вытяни ОДНУ самую честную строку из моего текста (или сформулируй её моими словами) — она будет всплывать на карточке. Не утешай, не лечи, оставь как есть.
3. Подбери палитру под настроение (тёплое/холодное/пустое/тревожное) — 3–4 цвета.
4. Сгенерируй один HTML-файл со следующим:
   - полноэкранный тёмный фон, генеративный визуал на Canvas 2D под это состояние (медленный, живой, не мельтешащий: частицы/волны/дыхание — что подходит настроению)
   - имя состояния и одна честная строка поверх, аккуратной типографикой
   - звук через Web Audio API (без сэмплов): непрерывный минорный эмбиент-пэд под настроение (тёплое — пышнее, пустое — суше и тише), запуск по клику
   - всё в одном файле, открывается двойным кликом, работает офлайн
5. Ничего лишнего: ни кнопок соцсетей, ни аналитики. Только карточка.

Верни готовый HTML целиком.
